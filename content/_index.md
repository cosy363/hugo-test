## OPMATE란? 

IT 인프라 운영 시 대량 단순 반복 작업을 자동으로 수행할 수 있는 **운영 자동화 플랫폼**입니다.

<br>


### 이렇게 활용하세요

*한 번의 명령으로 모든 서버의 디스크 사용률을 조사해 보세요.*

`$ opmate execution run -ts diskusage`

<br>

*소프트웨어 패치 파일을 일괄로 배포하세요.*

`$ opmate execution run -ts downpatch -ag 'patch.tar.gz'`
